# Product Overview

WealthFlow is an AI-powered personal finance management application that provides intelligent financial tracking and insights.

## Core Features

- **Budget Management**: Create and track budgets with categories, periods, and spending limits
- **Expense Tracking**: Monitor expenses across multiple categories (Food, Rent, Transport, Entertainment, Health, Utilities, Education, Shopping, Other)
- **Income Management**: Track income from various sources with recurring payment support
- **Savings Goals**: Set and monitor savings targets with progress tracking
- **AI Insights**: Automated financial recommendations and spending analysis
- **Transaction Management**: Comprehensive transaction history and categorization
- **Financial Analytics**: Charts, trends, and spending distribution analysis

## User Experience

- Dark mode by default with theme switching capability
- Responsive design for mobile and desktop
- Real-time data updates with optimistic UI patterns
- Interactive dashboards with charts and visualizations
- Command palette for quick actions

## Target Users

Individuals seeking intelligent financial management with automated insights and comprehensive tracking capabilities.
